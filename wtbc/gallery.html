Have photos to share? Send them to info@wtbc.co.uk and we'll add them to the gallery.

📷 How to add real photos:
  1. Put your image files (JPG/PNG) in the same folder as this website
  2. Open gallery.html in a text editor
  3. In the photos2025 array, change src: '' to src: 'your-photo.jpg'
  4. Update the caption to match
  5. Add more objects to the array for more photos
